# Bridge Studio — Portfolio

Brand Design & Art Direction studio portfolio built with Vite.

## Setup

```bash
npm install
npm run dev
```

## Deploy (Vercel)

Push to GitHub → Import on [vercel.com](https://vercel.com) → Auto-detect Vite → Deploy.

## Structure

```
bridge-studio/
├── index.html          # Main HTML
├── src/
│   ├── style.css       # All styles
│   └── main.js         # Canvas, i18n, SPA routing
├── public/
│   └── logo.svg        # Bridge logo
├── vite.config.js
├── vercel.json
└── package.json
```
